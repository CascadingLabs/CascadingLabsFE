# CascadingLabsFE Repo Orientation
Read `AGENTS.md` first, then inspect the relevant `src/`, `docs/`, or `public/` area.

Focus on:
- Astro + Starlight structure
- Solid interactive components
- Biome formatting rules
- reference-link rendering rules
- relevant package scripts in `package.json`

Output: task-relevant repo summary, likely files, commands, constraints, and open questions before editing.
