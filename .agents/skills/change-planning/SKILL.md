# CascadingLabsFE Change Planning
Plan against the smallest affected surface first: landing page, Starlight docs, shared components, Solid islands, styles, or plugins.

Include:
- intended visible outcome
- affected pages/components/docs
- SEO/reference-system implications if relevant
- non-goals
- risk areas
- verification needs
