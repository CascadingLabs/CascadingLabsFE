# CascadingLabsFE Verification Plan
Choose the smallest trustworthy evidence set.

Consider:
- `bun run check`
- `bun run build`
- targeted visual/manual review for page or theme changes
- `bun run lighthouse` when performance or page quality is in scope

Report behavior to prove, exact commands, manual checks, and remaining gaps.
